# Synthesis

## Objective

The RTL design of the Parameterized 16-bit ALU was synthesized using **Yosys**, an open-source synthesis tool. The objective was to verify that the RTL is synthesizable, optimize the design, and generate the gate-level representation.

---

## Synthesis Tool

* **Tool:** Yosys
* **Language:** Verilog HDL
* **Target Module:** `alu`
* **Design Type:** Parameterized 16-bit Arithmetic Logic Unit (ALU)

---

## Synthesis Command

```bash
yosys -p "read_verilog rtl/alu.v; synth -top alu; stat"
```

---

## Synthesis Outputs

The following files were generated during synthesis:

```text
synthesis/
├── synthesis_report.txt
├── alu_rtl.svg
└── alu_rtl.dot
```
